- Concurrencia y Arquitecturas Multiprocesador : Laboratorio
- Laboratorio : Implementación paralela del Algoritmo PSO.
- Arquitectura de Computadoras, Compiladores y Medidas de Desempeño : Teoría.
- Arquitectura de Computadoras, Compiladores y Medidas de Desempeño : Laboratorio.
